계산기탭: 계산기를 보여줌

사용법탭: 사용법을 보여줌

왼쪽 화면: 변수에 저장된 값 계산식등 계산과정을 보여주는 화면

오른쪽 화면: 값을 보기 편하게 나타내줌

아래화면: 누른 값이나 결과값을 나타내줌

저장버튼: 계산과정과 결과값을 저장

불러오기 버튼: 저장한 값을 불러옴

저장삭제버튼: 저장한 값을 삭제함

상단 +버튼: 글씨 크기를 키워줌

상단 -버튼 : 글씨 크기를 줄여줌

← : 한글자씩 지워줌

CL: 화면을 초기화 시켜줌

Enter: 입력
